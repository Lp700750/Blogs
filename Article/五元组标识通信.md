# 五元组标识通信
>![image](https://github.com/Lp700750/Blogs/assets/104414865/d4c1837b-867b-4e96-8a0d-0a062a6eea2a)
>在TCP/IP协议中，用"源IP"，"目的IP"，"源端口号"，"目的端口号"，"协议号"，这五元组来标识一个通信。
>- 端口号：标识另一台主机上的一个进程。
>- 端口号如何找到进程？内核实现通过哈希算法，端口号对应进程PID。
>- 源IP：发送数据的主机IP
>- 目的IP：最终接收数据主机IP
>- 源端口号：发送数据主机中发送数据的进程
>- 目的端口号：最终接收数据主机中要接收数据的进程。
