### 函数栈帧的创建与销毁  
#### 0 前言  
在学习编程语言的过程当中，函数是我们绕不开的一个话题，它通过复用帮助我们节省很多的代码，是一个非常好的模块。但是我们可能常常只知道它的表层使用炫酷，对于它的底层部分我们是否了解呢？今天就让我们走进函数栈帧的世界，揭开它那神秘的面纱。
#### 1 举一个加法函数栗子
```
#include<stdio.h>
int add(int a,int b)
{ 
  int sum=a+b;
  return sum;
}
int main()
{
  int a=10,b=20;
  int c=0;
  c=add(a,b);
  printf("a+b=%d",c);
  return 0;
}
```
